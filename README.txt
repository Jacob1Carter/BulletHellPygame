Controls:

    ESC: pause/play

    WASD: move

    Mouse: aim

    M1: shoot

    M2: dash

    SPACE: zoom

    E: shoot rocket

    Q: throw glaive

    F: place warp

    F (if warp is placed): activate warp

    R: use health pack

Reticule options can be found in the pause menu

Eliminate opponents to progress through the phases, after phase 5, beat the boss.
